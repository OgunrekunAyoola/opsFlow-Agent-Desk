# Deployment

OpsFlowAI can be deployed to your own infrastructure for maximum control and data privacy.

## Railway

The fastest way to deploy for small to medium teams.

<CodeBlock 
  language="bash" 
  code="railway up" 
/>

## Docker

Every OpsFlowAI release is published as a Docker image.

<CodeBlock 
  language="yaml" 
  code="services:
  opsflow-agent:
    image: opsflow/agent-desk:latest
    ports:
      - '3000:3000'
    environment:
      - DATABASE_URL=${DATABASE_URL}" 
/>
