# API Reference

Interact with OpsFlowAI programmatically via our REST API.

## Authentication

All API requests must include an `Authorization` header with your API key.

<CodeBlock 
  language="http" 
  code="Authorization: Bearer YOUR_API_KEY" 
/>

## Tickets

### List Tickets
`GET /api/tickets`

Returns a paginated list of tickets.

### Get Ticket
`GET /api/tickets/:id`

<ApiTable data={[
  { name: "id", type: "string", desc: "Unique ticket identifier.", required: true },
  { name: "status", type: "string", desc: "The current status of the ticket." },
  { name: "customerEmail", type: "string", desc: "The email of the customer." }
]} />
